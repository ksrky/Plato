# Installation

## Prerequisities

Plato is developed by [Stack](https://docs.haskellstack.org/en/stable/), so you need to install it before installing Plato.

## Getting the source

**From GitHub**

```command
$ git clone https://github.com/ksrky/Plato.git
```

## Building and Installing

**Using stack**

```command
$ cd Plato
$ stack install
```
